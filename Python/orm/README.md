# orm
模仿 GORM 的 Python ORM 工具，簡化 Python 與資料庫連線、下達指令等步驟，並且定義接口方便 TOML、YAML、JSON 設置連線資訊。

## 前置條件
- Python/SQLAlchemy 2.0.0+: 目前 Python 最推薦的 ORM 套件。
- 各式 Python 與 SQL 的連線套件，例如 oracledb、psycopg2-binary 等等，請查閱各資料庫接口.py。
